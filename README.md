# Bridge Addis - Internship Connector

## Project Description
Bridge Addis is a frontend web platform designed to connect university students in Addis Ababa with local businesses and NGOs. The goal is to help students transition from theoretical knowledge to real-world professional experience.

## Features
- **Student Registration**: Students can create profiles saved via LocalStorage.
- **Organization Portal**: Local businesses and NGOs can register their details.
- **Internship Board**: Organizations can post opportunities that students can browse.
- **Application System**: A functional bridge that allows students to apply for specific roles.
- **Responsive Design**: Fully functional on mobile, tablet, and desktop views.

## Technologies Used
- **HTML5**: Semantic structure for accessibility and SEO.
- **CSS3**: Custom styling using Flexbox, Gradients, and Transitions (No frameworks).
- **JavaScript (Vanilla)**: DOM manipulation, Event listeners, and LocalStorage for data persistence.

## How to Run the Project
1. Clone the repository: `git clone [YOUR_GITHUB_LINK_HERE]`
2. Open the folder in VS Code.
3. Right-click on `index.html` and select **"Open with Live Server"**.
4. Start by registering an Organization, then post an Internship!
